## :pencil: Descripció
Aquest repositori conté la **pràctica de gestió de projectes amb Git** de l'assignatura *Desplegament d'Aplicacions Web*.

L'objectiu de la pràctica és demostrar el domini de les eines bàsiques i avançades de **control de versions amb Git**, simulant un entorn de desenvolupament col·laboratiu, resolent conflictes de merge i treballant amb un repositori remot a GitHub.

---

## :open_file_folder: Contingut del projecte
El projecte inclou els següents elements principals:

- :elephant: **Aplicació PHP bàsica**  
  - Sistema simple de gestió de tasques
  - Sistema d'usuaris
- :herb: **Gestió de branques**
  - Branca principal `main`
  - Branques de funcionalitats (`feature/add-tasks`, `feature/add-user-system`)
- :twisted_rightwards_arrows: **Resolució de conflictes**
  - Merge amb conflicte resolt manualment
- :cloud: **Sincronització amb GitHub**
  - Ús de `fetch`, `pull` i `push`
- :recycle: **Gestió de canvis**
  - Ús de `git restore` per desfer canvis locals
  - Ús de `git revert` per revertir commits problemàtics
- :page_facing_up: **Documentació**
  - README.md
  - notes.txt amb diferències entre `restore` i `revert`
